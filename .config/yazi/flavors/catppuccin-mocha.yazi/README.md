## 🎨 Installation

```sh
ya pack -a yazi-rs/flavors:catppuccin-mocha
```

## ⚙️ Usage

Add these lines to your `theme.toml` configuration file to use it:

```toml
[flavor]
use = "catppuccin-mocha"
```
